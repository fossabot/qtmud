qtMUD is a Python package containing a MUD driver written by emsenn and released under the MIT License.

MUDs are a form of multi-user server where clients receive text output and interact by typing commands. MUDs are usually games, and are often role-playing games.

qtMUD is the engine, or driver, for running a MUD server, and is intended to be used with a MUD /library/: modules adding game features.

To learn more about the qtMUD project, read qtMUD Project Information.

To learn more about playing a game that uses a qtMUD library, read the List of Live Games.

To learn more about hosting your own game using a qtMUD library, read the List of qtMUD Libraries.

To learn more about developing a MUD library, read the Manual of qtMUD Library Development.

To learn more about developing qtMUD, read the Manual of qtMUD Development.
