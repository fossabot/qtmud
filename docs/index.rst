qtMUD is a Python package for developing and managing a multi-user dimension,
or :term:`MUD`.  It is developed by `emsenn <https://github.com/emsenn/>`_
and released under the `WTFPL <http://www.wtfpl.com>`_.

.. note:: qtMUD is just the core package for running a MUD.  If you aren't a 
          developer, you are unlikely to find it useful.
