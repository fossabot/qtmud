starhopper.builders module¶

build objects for in-game interaction

This module houses the methods for building things, and generating non-thing in-game objects. In this context, a thing refers to an instance of qtmud.Thing.

starhopper.builders.build_planet(basis: object = None, system=None, distance=None, points=None)[source]¶

creates a planet

Parameters:
  • basis – expected to be a qtmud.Thing
  • system – the system the planet will live in
  • distance – the distance from the system’s sun
  • points – how many points can be spent building the system.
Returns:

an object which behaves like a planet

Is basis is passed, the planet is built upon that object - which is expected to be an instance of qtmud.Thing. If basis is not passed, the method creates a new object with qtmud.new_thing().
