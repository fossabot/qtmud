qtmud.qualities.sighted module¶

Gives the ability to see to a thing.

New in version 0.0.2-feature/parser.

Adds sight to a quality, which for now means access to the look function and command.

class qtmud.qualities.sighted.Sighted[source]¶

Bases: object

Adds sight to a thing it is applied to.

New in version 0.0.2-feature/parser.

When applied by the manager, gives a thing sight. In this context, that means the look function, documented below.
