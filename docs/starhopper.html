STARHOPPER

New in version 0.0.4.

Starhopper is the first MUD library (mudlib) produced using qtmud. A simple action adventure game, Starhopper works best when played as a session-based game with a few friends.

Attributes:
SPLASH(str): What the player sees when they connect,
after qtmud’s own splash. Expected in every mudlib, but isn’t necessary.

START_LOCATION(object): first system, where new players start. players(list): all the connected players, referenced by

their class.

When this module is imported, it also adds every method in starhopper.subscriptions to qtmud.subscriptions.
